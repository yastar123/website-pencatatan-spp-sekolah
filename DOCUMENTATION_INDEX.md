### Beginner

2. Read [INSTALLATION.md](./INSTALLATION.md) - 10 min
3. Install & run locally
4. Try login & explore dashboard
5. Read relevant sections of [USER_GUIDE.md](./USER_GUIDE.md)

### Intermediate
1. Complete Beginner path
2. Read [USER_GUIDE.md](./USER_GUIDE.md) fully - 30 min

4. Generate reports & export data
5. Manage settings & pengaturan

1. Complete Intermediate path
2. Read [API_DOCUMENTATION.md](./API_DOCUMENTATION.md) - 20 min
3. Test APIs with cURL/Postman
4. Review source code in `/app`, `/components`, `/lib`
5. Build custom features or integrate with other systems
